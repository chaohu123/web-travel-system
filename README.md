## 个性化旅游路线规划与旅友结伴平台

本项目是一个 **前后端分离** 的 Web 系统，提供：

- 个性化旅游路线智能规划
- 旅友结伴匹配与小队管理
- 旅行游记 / 攻略分享与社区互动

技术栈：

- 后端：Spring Boot 3 + Spring Security + JPA + MySQL + Redis
- 前端：Vue 3 + Vite + TypeScript + Pinia + Vue Router + UI 组件库（Element Plus / Ant Design Vue）

目录规划（后续逐步完善）：

- `backend/`：后端 Spring Boot 服务
- `frontend/`：前端 Vue 3 应用
